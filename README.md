Guideline

Professional Business Presentation – Font & Format Specification

1. Slide Size & Layout
Slide Size:
- Widescreen 16:9
- Resolution: 1920 × 1080 px

Margins (Safe Area):
- Left / Right: 1.2 cm
- Top: 1.0 cm
- Bottom: 1.0 cm

Alignment:
- All text left-aligned
- Center alignment only for cover slide


2. Font Specification (Corporate Standard)

Primary Font (Recommended):
- Inter
(Alternatives: Segoe UI, Calibri, Montserrat)


Title / Heading:
- Font: Inter SemiBold
- Font Size:
  - Cover Title: 40–44 pt
  - Section Title: 32 pt
  - Slide Title: 28 pt
- Line spacing: 1.1
- Letter spacing: Normal
- Case: Title Case (avoid ALL CAPS)


Subtitle:
- Font: Inter Medium
- Font Size: 18–20 pt
- Line spacing: 1.2
- Color: Secondary Grey


Body Text:
- Font: Inter Regular
- Font Size: 16–18 pt
- Line spacing: 1.3–1.4
- Maximum bullets per slide: 5
- Bullet style: Standard dot


Caption / Footnote:
- Font: Inter Regular
- Font Size: 10–11 pt
- Color: Grey 60%


3. Color Palette (Corporate Safe)

Primary Palette Example:
- Primary (Title): #0F172A – Dark Navy
- Secondary (Body): #334155 – Slate Grey
- Muted Text: #64748B

Accent Colors (Use minimally):
- Accent Blue: #2563EB
- Accent Teal: #0D9488
Maximum one accent color per deck


4. Background Rules
- Default background: White (#FFFFFF)
- Cover & Section Divider:
  - Solid dark background or subtle gradient only

Avoid:
- Busy patterns
- Excessive background images


5. Bullet & Text Rules
- Bullet indentation: 0.6 cm
- Sub-bullet indentation: 0.3 cm
- Use concise statements, not long paragraphs


6. Divider & Line Usage
- Line thickness: 0.5–1 pt
- Color: Grey 30%
- Used to separate title and content or highlight sections


7. Chart & Data Formatting

Chart Font:
- Same as body font
- Axis title: 12 pt
- Axis label: 11 pt
- Data label: 11–12 pt

Chart Color Rules:
- Primary data: Primary color
- Secondary data: Grey
- Highlight: Accent color

Avoid:
- 3D charts
- Heavy shadows
- Thick gridlines


8. Icons & Visuals
- Style: Outline / Line icons
- Color: Same as text color
- Size: 16–20 px
- Use one consistent icon style only
- Avoid icons if not necessary


9. Section Divider Slide Format
- Title font size: 36 pt (White)
- Subtitle font size: 16 pt (Light Grey)
- Background: Dark Navy / Charcoal


10. Footer (Optional)
- Left: Company name / Presenter
- Right: Slide number
- Font size: 10 pt
- Color: Grey 50%


11. Animation & Transition
- Transition: None or Fade (0.2s)
- Animation: None (recommended)


12. File Naming Convention
ClientName_ProjectName_Presentation_v1.0.pptx
